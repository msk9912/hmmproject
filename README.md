# 🎬 Local Shorts’ (2022.03 ~ 2022.10) 
위치 기반 SNS 플랫폼

## 💻 프로젝트 소개
Local Shorts'는 위치를 기반으로 한 sns플랫폼입니다. 
1분 이내의 짧은 세로형 영상을 지도 속 특정 장소를 직접 지정해 업로드하며 
*사운드태그기능을 이용해 gps로 구분되지 않는 건물 내부(특정 강의실 등)에
직접 방문시에만 업로드 및 확인 가능한 영상 목록들을 제공합니다.

##### *사운드태그기능(비가청주파수를 활용한 위치 구분 기능)

  

## 📂   프로젝트 기능
- 로그인, 회원가입, 마이페이지
- 해시태그 및 해시태그 검색 기능
- 리액트를 활용한 웹 페이지 구현
- 사운드태그(주파수 인식 기반 위치 탐지) 기능 구현
- GoogleMap API 연동 및 지도 데이터 처리
- 장소별 영상 업로드, 공개범위 설정, 해시태그 기능 구현
- 피드, 마이페이지 CRUD
- 영상 스트리밍 기능 구현, 영상 목록(DB) MySQL연동
